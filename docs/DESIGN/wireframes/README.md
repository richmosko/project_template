# Wireframes

Low-fidelity layouts — structure before style. One file per key screen.

- **Formats:** PNG/SVG exports, or plain HTML wireframes. Keep them lightweight.
- **Naming:** `<screen>-<NN>.png` (e.g. `onboarding-01.png`, `dashboard-01.png`).
- **Reference** them from `../index.html` § Wireframes with relative paths: `<img src="wireframes/onboarding-01.png">`.
- Promote anything that will ship to Figma (via the `ux-designer`'s Figma skills); keep the exported still here for in-repo review.
