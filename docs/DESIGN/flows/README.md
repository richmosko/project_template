# User Flows

How users move through the product to complete the highest-traffic stories.

- **Formats:** Mermaid `flowchart` in `.md`/`.mmd` files (preferred — diffable), or exported images.
- **Naming:** `<task>-flow.md` (e.g. `signup-flow.md`, `checkout-flow.md`).
- **Reference** them from `../index.html` § User Flows. Inline first-pass flows can live directly in `index.html` as `<pre class="mermaid">` blocks; promote complex ones to files here.
- Start with the top 3–8 PRD stories — one flow per major task.
